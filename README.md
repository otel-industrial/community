# OpenTelemetry Industrial Community

A community for practitioners working on OpenTelemetry in industrial and legacy environments. OpenTelemetry has made enormous strides in cloud-native observability — but industrial protocols, legacy constraints, and OT/IT convergence remain largely unexplored territory. This is where that work happens.

## Get Involved

### Slack

Join the conversation on the [CNCF Slack](https://slack.cncf.io/) in the **#otel-industrial** channel.
